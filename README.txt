Instruções

1) Para executar o programa é necessário mudar o caminho de salvamento dos arquivos tanto no servidor quanto no cliente. 
server.py = Linhas 40 e 112 do código
client.py = Linhas 32 e 133 do código

2) A classe Pacote.py deve estar presente tanto do lado do servidor quanto do lado do cliente na execução.

3) O cliente deve ter uma pasta na chamada "client" no local de execução. Essa pasta contém os arquivos passíves de serem enviados ao servidor.

4) O servidor deve ter uma pasta na chamada "server" no local de execução. Essa pasta contém os arquivos passíves de serem enviados ao cliente.

5) Para o cliente ser executado, o server deve estar em execução.
